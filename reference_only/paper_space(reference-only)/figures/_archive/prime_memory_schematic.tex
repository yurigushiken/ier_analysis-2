\caption{Prime memory classification analysis. Each column shows a trial pair
sharing the same target but differing in prime identity. The classifier
attempts to decode which prime preceded trials with identical targets. Blue
borders indicate PI-range primes (1-4); red borders indicate ANS-range primes
(5-6); targets are grayscale. Columns illustrate three comparison types:
Within-PI (both primes from PI range), Cross-System (one PI, one ANS prime),
and Within-ANS (both primes from ANS range). If PI and ANS encode numerosity
differently, cross-system pairs should be more discriminable than within-system
pairs. (Stimuli colored for illustration; actual stimuli were grayscale.)}
